Verifiable Voice Protocol (VVP) is a cryptographic framework for establishing trust in voice communications. It enables verifiable caller identity using KERI-based credentials and PASSporT tokens.

What We Do

Verify Identities

Validate PASSporT tokens and their associated credential chains. Confirm that callers are who they claim to be using cryptographic proofs.

Create Dossiers

Issue and assemble ACDC credentials into dossiers that prove organizational identity, role assignments, and telephone number allocations.

Learn about creation

Key Concepts

PASSporT
A signed JSON Web Token (JWT) containing caller and callee identities, with an evidence URL pointing to the credential dossier.
ACDC
Authentic Chained Data Container - a KERI credential format that supports cryptographic binding, selective disclosure, and revocation.
Dossier
A bundle of ACDC credentials forming a trust chain from trusted roots (like GLEIF) to the entity making the call.
KERI
Key Event Receipt Infrastructure - a decentralized key management system providing secure identifier rotation and delegation.
